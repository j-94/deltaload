INSTALL
-------

[](https://github.com/BigBrotherTrade/trader?screenshot=true#install)

先装talib的C库，再用pip装依赖。

修改 mysql 配置文件 /etc/my.cnf.d/server.cnf ，增加如下配置:

> \[mysqld\]
> 
> wait\_timeout=31536000
> 
> interactive\_timeout=31536000

重启mysql服务
