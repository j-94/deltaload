FAQ
---

[](https://github.com/Semantic-Org/Semantic-UI-React?screenshot=true#faq)

**Can I use custom Icons?** Yes. Just use `<Icon className='my-icon' />` instead of `<Icon name='my-icon' />`. See [#931 (comment)](https://github.com/Semantic-Org/Semantic-UI-React/issues/931#issuecomment-263643210) for detailed info and examples. **How do I setup CSS?**There are several options. Refer to our doc on [CSS Usage](https://react.semantic-ui.com/usage#css).

**Can I use a custom CSS theme?** Yes. Semantic UI React includes components that render valid Semantic UI HTML, no CSS is included. This allows you to load any Semantic UI CSS theme on top of your Semantic UI React app.  
Here are some helpful links:

*   [Building CSS with Webpack](https://medium.com/webmonkeys/webpack-2-semantic-ui-theming-a216ddf60daf)
*   [Building CSS with Meteor](https://github.com/Semantic-Org/Semantic-UI-Meteor)
*   [Creating a standalone theme](http://learnsemantic.com/themes/creating.html)
